Golden Gate Bakery API
===========

Light-weight API to check if the [Golden Gate Bakery](http://goldengatebakery.com/) is open.

## Endpoint:

http://dantat.herokuapp.com/api/isopen/

## Request:
```bash
curl -X POST
     -H "Content-type: application/json"
     http://dantat.herokuapp.com/api/isopen
```

## Response:
```javascript
false
```
