Dantat

A simple API to determine whether or not Golden Gate Bakery is open.


Sample request

  curl http://dantat.herokuapp.com/api/isopen

Sample responses

If the bakery is open, the response will be 

  {
    message: true,
    status: 200
  }

If the bakery is closed, the response will be 

  {
    message: false,
    status: 200
  }
